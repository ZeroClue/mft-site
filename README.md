# MFTPlus Landing Page

Product landing page for [mftplus.co.za](https://mftplus.co.za). Built with React + Vite + TypeScript.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Auto-deploys to Vercel from the `main` branch.
